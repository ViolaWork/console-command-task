# console-command-task
a Python console program that accepts one command line argument: a number value that is used as initial value of a result variable. The program prompts its user for commands in an infinte loop. After each prompt, it applies the command and prints out the new result value.

The program is writen with python so it Require Python 3.x inistalled in your system

How to Run the program:
1- open the terminal 
2- run this: python main.py <initial_value> 
 you have to insert an integer number as initial value inplace of <initial_value>  
for example: python main.py 4

Commands that you can use:
increment: Increase result by 1
decrement: Decrease result by 1
double: Double the result
randadd: Add a random number
undo: Undo the last command
exit: Stop the program

Assumptions:
initial value is an integer number
randadd adds a random integer between 1 and 10
